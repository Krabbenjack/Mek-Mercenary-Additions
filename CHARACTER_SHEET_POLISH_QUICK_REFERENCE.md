# Character Sheet Visual Polish - Quick Reference

## Before vs After Summary

### Typography Changes

#### Labels (Supporting Information)
```
BEFORE: Size 9, Bold, #1E1E1E (dark)
AFTER:  Size 8, Normal, #666666 (light gray)
RESULT: Less visually dominant, clearly supporting
```

#### Values (Primary Information)
```
BEFORE: Size 9, Normal, #1E1E1E
AFTER:  Size 10-12, Bold/Normal, #1E1E1E
RESULT: More prominent, easier to read at a glance
```

#### Stat Blocks (Attributes, Progress Bars)
```
BEFORE: Size 9, Normal, mixed hierarchy
AFTER:  Size 12, Bold, Centered
RESULT: Clear visual anchors, "printed sheet" stat blocks
```

### Spacing Changes

#### Row Spacing
```
BEFORE: 1-3px between items
AFTER:  3-6px between items
RESULT: Better breathing room, cleaner rhythm
```

#### Section Padding
```
BEFORE: 0-10px margins
AFTER:  8-16px margins
RESULT: Content has space to breathe, less cramped
```

#### Content Areas
```
BEFORE: Direct packing to body
AFTER:  Content frames with 8px padding
RESULT: Visual buffer, more professional
```

## Visual Design Goals

### 1. Stat Blocks as Anchors ✅
- **Goal**: Important numbers readable at a glance
- **Method**: Larger (12pt), bold, centered
- **Example**: Attribute values, XP, Progress values

### 2. Labels Never Compete ✅
- **Goal**: Supporting info doesn't distract from values
- **Method**: Smaller (8pt), lighter color (#666666)
- **Example**: "STR:", "Confidence:", "Rank:"

### 3. Paper Sheet Rhythm ✅
- **Goal**: Consistent, predictable spacing
- **Method**: Standard spacing scale (3-6px rows, 8-16px margins)
- **Example**: All sections feel balanced

### 4. Calm Aesthetic ✅
- **Goal**: Reduced visual noise
- **Method**: Lighter colors for supporting text, more whitespace
- **Example**: Less bold, more breathing room

## Color Palette

| Element Type | Color | Purpose |
|-------------|-------|---------|
| **Values** | #1E1E1E | Primary information (dark, readable) |
| **Labels** | #666666 | Supporting text (light gray, recedes) |
| **Headers** | #444444 | Section titles (medium gray) |
| **Hints** | #888888 | Supplementary info (lighter gray) |

---

## Spacing Scale

| Type | Value | Usage |
|------|-------|-------|
| **XS** | 3-4px | Tight row spacing |
| **SM** | 5-6px | Standard row spacing |
| **MD** | 8px | Content padding |
| **LG** | 12px | Section padding |
| **XL** | 16px | Major section separation |

---

## Font Scale

| Size | Weight | Usage |
|------|--------|-------|
| **12pt** | Bold | Stat values (attributes, XP) |
| **11pt** | Bold | Skill levels |
| **10pt** | Normal/Bold | Character values, SPA names |
| **9pt** | Normal/Bold | Normal text, section headers |
| **8pt** | Normal/Italic | Labels, hints, subsection headers |

## Result

The Character Sheet now has:
- ✅ Clear visual hierarchy (values prominent, labels supporting)
- ✅ Readable stat blocks (numbers stand out)
- ✅ Generous spacing (breathing room everywhere)
- ✅ Consistent rhythm (predictable layout)
- ✅ Calm aesthetic (reduced visual noise)
- ✅ Paper sheet feel (like a form to fill out)

**Feels like**: A printed Pen & Paper character sheet
**NOT like**: A business app, dashboard, or data editor
